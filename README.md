# pywiz
"pywiz" - a Python Web Game that combines education and entertainment to make learning Python syntax an engaging experience. Leveraging the power of Flask, React, Pygame, and MongoDB, pywiz offers a dynamic platform where players can enhance their Python skills through interactive gameplay

FEATURES-

1. Provide an interactive and entertaining environment for players to learn and practice
Python syntax.
2. Enhance players' understanding of Python concepts through a fill-in-the-blanks
gameplay approach.
3. Offer real-time feedback and progression tracking to encourage continuous learning
and improvement.
4. Increase accessibility to Python education by making learning materials readily
available in a gamified format.
5. Foster a sense of accomplishment and motivation by incorporating levels, lives, and
unlockable content to incentivize progression.


Tools & Technologies used-
Tools: Visual Studio Code, Postman, Thunderclient, PyCharm, Git
Technologies: Python, npm
Libraries: Flask, React, Pygame, MongoDB, pipenv, pymongo, requests, bcrypt, random, json
